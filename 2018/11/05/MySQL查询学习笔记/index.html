查询语句的使用

查询所有字段

  • select * from 表名;

查询指定字段

  • select 列1,列2,… from 表名;

使用as给字段起别名

  • select 字段 as 名字 … from 表名;

  • select 表名.字段 … from 表名

  • 可以通过as给表起别名

  • select 别名.字段 … from 表名 as 别名;

  • 消除重复行

  • distinct 字段

条件查询

  • 比较运算符

  • select … from 表名 where …;

  • 逻辑运算符

  • and

  • or

  • not

模糊查询

  • like
  • %替换0个或者多个

  • _替换1个

  • rlike 正则表达式

范围查询

  • in (1,3,8)表示在一个非连续的范围内

  • not in 不在此非连续范围内

  • not between … and …表示不在一个连续的范围内

  • 空判断

  • is null

  • is not null

排序

  • order by 字段
  • asc从小到大排序，即升序
  • desc从大到小排序，即降序

聚合函数

  • count 总数

  • max 最大值

  • min 最小值

  • sum 求和

  • avg 平均值

  • round 四舍五入 round(123.12,1)保留一位小数123.1

分组

  • group by

  • group_concat(…)

  • having

分页

  • limit start,count
